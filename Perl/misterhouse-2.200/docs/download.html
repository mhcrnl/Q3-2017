Where to Download the latest version of MisterHouse

Version 2.105 released on 12/01/2008:

  • If using a Windows:
    • misterhouse_src_2105.zip (previous stable release: 2.104): The Perl source files and documentation. About 12 Meg. If you have Perl installed, this is all you need.
    • misterhouse_win_2105.zip (previous stable release: 2.104) : The Windows compiled file. About 6 Meg. This will run stand alone without Perl installed on Windows 95,100, NT, or 2K. Note: the above misterhouse_src_2105.zip is also required, as it has documentation and support files.
  • If using a Unix, the tar file is about 9 Meg:
    • misterhouse-2.105.tar.gz (previous stable release: 2.104)

Between formal releases, you can pick up the latest updates from the SVN repository (more info on our wiki ) Changes in the test release are listed in the SVN log.

You can get older versions of MisterHouse on the Sourceeforge download page

A large (30 meg) file of optional files (mainly sound files) is here: misterhouse_misc.zip


Subscribe to the misterhouse-anounce mailing list to get an email summarizing changes when a new release is available.
