clickTk - version 4.1 Changes

clickTk - version 4.11 Changes

  - Preview shows all widgets of DialogBox.
  - Support argument list to the composite constructor.
  - Order accepts any statements.
  - Label text may accept parenthesis.
  - Allow define composite widget from user libraries 
    (new option 'pathName' for widget class definition).
  - Support scrolled Frame and Pane.
  - Compatibility fix for treewalk value (assume 'D').
  - Improved handling of non-visual Widget.
  - New option type 'List', i.e. for -labelpack options list.
  - Allow double quotations on widget options.
  - Improved session flow
  - allow restart session
  - Some minor bugs fixed.
